# Smart LED Control System

## Objective
Build a mini project combining LED control, button input, and serial monitoring.

## Components
- ESP32
- LED
- Push Button

## Concepts Covered
- Digital Input
- Digital Output
- Serial Communication
- State Control

## Working
The button toggles the LED state while the Serial Monitor displays current system status.

## Learning Outcome
Applying all Module 1 concepts in a single project.
