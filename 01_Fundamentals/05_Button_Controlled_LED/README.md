# ESP32 Button Controlled LED

## Objective
Control an LED using a push button.

## Components
- ESP32
- Push Button
- LED
- Resistor

## Concepts Covered
- Digital Input
- Digital Output
- Conditional Statements

## Working
Pressing the button turns the LED ON. Releasing the button turns the LED OFF.

## Learning Outcome
Understanding interaction between inputs and outputs.
