# ESP32 Reaction Timer Game

## Objective
Measure user reaction time using a button and LED.

## Components
- ESP32
- LED
- Push Button

## Concepts Covered
- Digital Input
- Digital Output
- Timing Functions
- User Interaction

## Working
The LED turns ON after a random delay. The user presses the button as quickly as possible, and reaction time is displayed.

## Learning Outcome
Combining multiple fundamentals into a mini application.
