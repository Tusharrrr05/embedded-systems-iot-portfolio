# ESP32 LED Blink Patterns

## Objective
Create different LED blinking patterns using timing control.

## Components
- ESP32
- LED
- 220Ω Resistor

## Concepts Covered
- Digital Output
- Timing Control
- Pattern Generation

## Patterns
1. Fast Blink
2. Slow Blink
3. Double Blink

## Learning Outcome
Understanding how timing affects digital outputs and pattern generation.
