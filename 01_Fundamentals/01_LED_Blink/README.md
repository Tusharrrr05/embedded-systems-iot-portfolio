# ESP32 LED Blink

## Objective
Learn the basic structure of an Arduino program by blinking an LED connected to ESP32.

## Components
- ESP32
- LED
- 220Ω Resistor

## Concepts Covered
- pinMode()
- digitalWrite()
- delay()
- GPIO Output

## Working
The ESP32 turns the LED ON and OFF repeatedly with a fixed delay.

## Learning Outcome
Understanding digital output control and Arduino program structure.
