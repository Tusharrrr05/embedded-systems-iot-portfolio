# Analog and PWM

## Overview

This section introduces analog signal processing, ADC (Analog-to-Digital Conversion), PWM (Pulse Width Modulation), and servo motor control using Arduino and ESP32.

## Topics Covered

- Analog Input
- ADC
- PWM
- Servo Motors
- Sensor Calibration

## Projects

- LDR Monitor
- LDR Controlled LED
- Potentiometer ADC
- PWM LED Fading
- Servo Motor Control
- Servo Control With Potentiometer

## Learning Outcomes

- Reading analog sensor values
- Generating PWM signals
- Controlling LED brightness
- Controlling servo motor position
- Understanding ADC and PWM concepts
