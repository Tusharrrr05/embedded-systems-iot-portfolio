# ESP32 Ultrasonic Distance Meter

## Objective
Measure distance using an HC-SR04 ultrasonic sensor.

## Components
- ESP32
- HC-SR04 Sensor

## Concepts Covered
- Pulse Timing
- Distance Calculation
- Sensor Interfacing

## Working
The sensor emits ultrasonic waves and calculates distance based on echo time.

## Learning Outcome
Understanding distance measurement systems.
