# ESP32 LDR Analog Read

## Objective
Read analog light intensity values from an LDR sensor using ESP32.

## Components
- ESP32
- LDR
- 10kΩ Resistor

## Concepts Covered
- Analog Input
- ADC Conversion
- Sensor Reading

## Working
The ESP32 continuously reads the LDR value and displays it on the Serial Monitor.

## Learning Outcome
Understanding analog sensor interfacing with ESP32.
