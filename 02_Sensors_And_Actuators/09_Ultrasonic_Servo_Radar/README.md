# ESP32 Ultrasonic Servo Radar

## Objective
Build a radar-like scanning system using a servo motor and ultrasonic sensor.

## Components
- ESP32
- HC-SR04
- Servo Motor

## Concepts Covered
- Sensor Integration
- Servo Sweeping
- Distance Measurement

## Working
The servo continuously scans while the ultrasonic sensor measures surrounding distances.

## Learning Outcome
Combining sensors and actuators into a practical project.
