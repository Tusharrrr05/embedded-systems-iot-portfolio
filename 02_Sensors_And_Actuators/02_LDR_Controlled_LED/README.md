# ESP32 LDR Controlled LED

## Objective
Automatically control an LED based on ambient light intensity.

## Components
- ESP32
- LDR
- LED
- Resistor

## Concepts Covered
- Analog Input
- Conditional Logic
- Automatic Control

## Working
The LED turns ON in darkness and OFF in bright conditions.

## Learning Outcome
Creating a basic automatic lighting system.
